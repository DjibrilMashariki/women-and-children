## üéì KNOWLEDGE BASE

### Implementation Pattern
All updated components follow this consistent structure:

```tsx
// Light mode classes (pre-existing)
className="bg-white text-gray-800 border border-gray-200"

// Add dark mode support
className="bg-white text-gray-800 border border-gray-200
           dark:bg-neutral-800 dark:text-white dark:border-neutral-700"

// Include hover states
className="hover:shadow-lg hover:bg-gray-50
           dark:hover:shadow-xl dark:hover:bg-neutral-700"
```

### Color Decision Logic
1. **Text**: Choose from 4 levels (white, neutral-100, neutral-300, neutral-400)
2. **Backgrounds**: Choose from 3 levels (neutral-900, neutral-800, neutral-700)
3. **Accents**: Consistent orange (#fbbf24 ‚Üí `dark:text-orange-400`)
4. **Borders**: Align with background level
